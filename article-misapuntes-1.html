Apuntes Sobre HTML

Atributos data-cualquiercosa

Este tipo de atributos se añadieron con la versión de HTML 5.
Los atributos data se usan para definir atributos que se usarán desde Javascript ya que el motor de renderizado del navegador no los va a tener en cuenta. Gracias a ellos, por ejemplo podemos tener un enlace así: 

<a href="mienlace.html" data-thumb="asset/thumb/imagen-1.png">Ver buena foto</a>
Y con un poco de Javascript podríamos hacer que al pasar por encima nos mostrase la foto que le pasamos con data-thumb.

Apuntes Sobre Javascript

Añadir eventos con addEventListener

La función addEventListener nos permite asociar funciones (handlers) a eventos que producen los elementos a los que se asocian.
Por ejemplo:

closeButton.addEventListener('click', function(event){console.log(this)});
Es importante tener en cuenta que el this dentro del la función del evento hace referencia al elemento que tiene el evento asociado, en este caso al link.
Se pueden añadir también eventos a la window por lo que por ejemplo si añadimos un handler al click, se disparará sea cual sea el sitio que cliquemos.

Funciones querySelector y querySelectorAll

Estas dos funciones son muy utiles para navegar por el DOM de nuestra aplicación.
querySelector(selectores) busca y devuelve un elemento que concuerde con los selectores pasados. El formato de los selectores es el mismo que para CSS.
Hay que tener en cuenta que si hay mas de un resultado nos devolverá el primero que encuentre. Null si no encuentra ninguno.
querySelectorAll(selectores) funciona igual que el primero pero devuevle una lista de nodos del DOm (NodeList).

Ejemplos: 

                        
//El primer elemento que tenga la clase 'featured'
const featured =  document.querySelector('.featured');

//El primer elemento que tenga la clase 'child' y esté dentro del elemento featured
const child =  featured.querySelector('.child');

//Un listado con todos los nodos que tengan la clase 'children'
const children =  document.querySelector('.children');
